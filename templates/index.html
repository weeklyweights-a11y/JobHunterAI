Employment type (LinkedIn)

Select one or more. With Internship checked, experience words are not added to keywords and LinkedIn’s experience filter (f_E) is omitted. For full-time only, experience level applies as usual.

Sources

Uses each job page’s posted date (fetched after Google finds links). 0 = no age filter. Separate from URL dedup below.

Each page is up to ~10 links. Stops early if a page adds no new URLs. Higher = slower and more CAPTCHA risk.

If Google shows a CAPTCHA, the run waits this long for you to solve it in the browser. If it is still blocked after that, ATS stops searching Google but keeps every job URL found so far and still merges them into the same Excel report as LinkedIn and other sources.

      Browser (same Chrome = new tabs)

      If this field is empty, JobHunter opens a second browser window (Playwright Chromium). That is expected.

      To reuse this Chrome (automation opens new tabs here, not another app): (1) Quit every Chrome window. (2) Start Chrome with debugging, e.g. "C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222. (3) In that window only, open this dashboard. (4) Paste http://127.0.0.1:9222 above, Save settings, then Start Hunting.
